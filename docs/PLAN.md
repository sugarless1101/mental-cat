# mental-cat PLAN (制作展まで)

## 現状
- 気分選択ボタン（good/neutral/bad）
- AIチャット（気分に連動して猫が話しかける）
- TASK UI（チェック可能だが保存/生成なし）
- SNS（YouTube埋め込み固定、SNSリンク）

## MVP（最優先 / 4日で完成させる）
### MVP1: ログイン + メモリ
- Laravel Breezeでログイン
- ログインユーザーごとにデータを保存・分離（user_idスコープ必須）
- 保存対象: 気分ログ / チャット要点 / タスク

### MVP2: チャット ↔ TASK 連動（見せ場）
- 会話に応じてAIがタスクを最大3つ提案し、TASKリストへ自動反映
- 「やったよ/終わった/できた」等の発言でタスクを自動で完了（最大1件）
- 手動チェックはconfirm → 定型の褒め言葉（API不要）

### MVP3: 画面の動き（軽量）
- 猫の簡易アニメ（CSSで揺れる/瞬き/浮遊など）※歩き回りは沼なら延期
- 手書き気分ボタンは別途（自分で実装）

## 追加（余裕があれば）
- BGM動的リコメンド（固定YouTube候補からbgm_keyで埋め込み切替）
- 気分推移グラフ（7日折れ線）
- 会話の積極性（5分後に1回だけ追いメッセージ）※沼なら延期

## スコープ方針
- AIは診断/治療を行わない。一般的で安全なセルフケアのみ。
- まず「動く」「データが残る」「見せ場がある」を優先する。
